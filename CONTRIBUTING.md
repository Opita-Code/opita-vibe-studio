# Contribuir a Vibe Studio

¡Gracias por tu interés! Vibe Studio es un proyecto independiente nacido en Colombia 🇨🇴 y toda contribución es bienvenida.

## Antes de empezar

1. **Fork** el repo y clona tu fork
2. Crea una rama descriptiva: `feat/mi-feature` o `fix/mi-bug`
3. Instala dependencias: `npm install`

## Workflow

```bash
npm run dev           # Levanta el frontend
npm test              # Corre tests antes de abrir PR
npm run typecheck     # Valida tipos
npm run lint          # Lint
```

## Convenciones

- **Commits**: Usa [Conventional Commits](https://www.conventionalcommits.org/) — `feat:`, `fix:`, `docs:`, `chore:`
- **PRs**: Describe qué cambiaste y por qué. Si hay UI, incluye screenshot.
- **Tests**: Si tocas lógica, agrega o actualiza tests.
- **Idioma**: El código y comentarios pueden estar en inglés o español. La UI está en español.

## ¿Qué puedo contribuir?

- 🐛 Bug fixes
- 📖 Documentación y traducciones
- 🎨 Mejoras de UI/UX
- 🧪 Tests
- 🌐 Nuevos proveedores de IA

## Reportar bugs

Abre un [issue](https://github.com/Opita-Code/opita-vibe-studio/issues) con:
- Descripción clara del problema
- Pasos para reproducir
- Screenshots si aplica

## Código de conducta

Sé respetuoso. No toleramos acoso, discriminación ni comportamiento tóxico. Estamos aquí para construir y aprender juntos.
