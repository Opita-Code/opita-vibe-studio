Guía de instalación en Windows

Si Windows SmartScreen muestra un aviso al abrir el instalador, sigue estos 2 pasos. Toma menos de 10 segundos.

¿Por qué aparece este aviso?

Windows muestra este aviso con todas las aplicaciones nuevas que no tienen un certificado de firma digital. Este certificado cuesta más de $400 USD al año.

Vibe Studio es un proyecto independiente y de código abierto — cada línea es verificable. Los instaladores se compilan automáticamente en GitHub Actions sin intervención humana. No hay nada oculto.

1
Haz clic en "Más información"
Windows mostrará una ventana azul. Busca el enlace y haz clic.
2
Haz clic en "Ejecutar de todos modos"
Aparecerá un botón nuevo. Haz clic y la instalación continuará.
🎉

¡Listo!

La instalación continuará normalmente. Bienvenido a Vibe Studio.

¿Usas macOS? Instrucciones aquí

Si Gatekeeper bloquea la app en macOS:

  1. Abre Preferencias del Sistema → Privacidad y Seguridad
  2. Busca el mensaje sobre Vibe Studio y haz clic en "Abrir de todos modos"
  3. Confirma en el diálogo que aparece
